# Real-Time-Object-Detection
Visionary: Real-Time AI Object Detection Dashboard

Visionary is a lightweight, web-based computer vision application that performs real-time object detection using the **YOLOv8** (You Only Look Once) architecture and the **Flask** web framework. 

This project transforms a standard webcam feed into an intelligent stream capable of identifying 80+ distinct object classes (such as people, laptops, cell phones, and chairs) with high efficiency, optimized for local machine performance.

---

## Features
* **Real-Time Inference:** Low-latency object detection using the YOLOv8-Nano model.
* **Web-Based Interface:** Clean, responsive frontend built with Bootstrap.
* **Live Streaming:** Efficient MJPEG streaming from Python/OpenCV to the browser.
* **Cross-Platform:** Specifically optimized for macOS (MacBook Air) and Windows.

---

## Tech Stack
* **Backend:** Python, Flask
* **Computer Vision:** OpenCV, Ultralytics (YOLOv8)
* **Frontend:** HTML5, CSS3 (Bootstrap 5)
* **Model:** PyTorch (`.pt`)
